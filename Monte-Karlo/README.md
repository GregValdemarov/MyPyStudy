# Моделирование системы массового обслуживания с непуассоновским потоком событий

## Исходный файл `monte-karlo.ipynb`

Код написан с помощью <a href="https://jupyter.org/">Jupyter Notebook</a> на языке `Python` версии `3.13.9`.

Код возможно запустить без использования `Jupyter Notebook` через любой предпочитаемый текстовый редактор, для этого необходимо последовательно скопировать и перенести в Ваш файл с кодом все содержимое блоков с кодом из файла `monte-karlo.ipynb`.
**Однако рекомендуется работать с фалом через `Jupyter Notebook` для удобства эксплуатации, т.к. код не оптимизировался под комфортную работу через запуск в текстовом редакторе**

Основные пояснения по постановке задачи, а также эксплуатации программы написаны в текстовых блоках в файле `monte-karlo.ipynb`.
Дополнительные пояснения к коду написаны в блоках с кодом в формате комментариев.

Результаты работ блоков кода файла `monte-karlo.ipynb` выводятся после самих блоков (если код предполагает выводы). В случае запуска кода через текстовый редактор, рузельтаты работы кода выводятся в консоль.

## Перечень использованных библиотек

- NumPy: 2.3.1
- Pandas: 2.3.0
- Matplotlib: 3.10.6
- SciPy: 1.16.2
